# Product K&W

Ce dossier documente l'orientation du projet **Knight & Wizard** dans ce depot :

- corpus source des regles et catalogues ;
- assistant MJ et joueur tabletop-first ;
- moteur multi-arbitre humain / joueur / LLM / auto ;
- architecture LLM et regles vivantes.

Le projet **K&W-game** est un projet separe, dans un autre depot, avec ses propres decisions produit et techniques. Les documents de ce dossier ne remplacent pas les decisions de K&W-game et ne doivent pas etre interpretes comme une spec du CRPG tactique coop.
