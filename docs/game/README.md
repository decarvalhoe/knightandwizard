# K&W-game

Ce dossier contient uniquement des documents de cadrage pour **K&W-game**, projet separe du present depot.

Le depot actuel `knightandwizard` reste le **corpus source K&W** : regles canonisees, catalogues, carte, donnees legacy, assistant MJ/Joueur tabletop-first et architecture LLM.

Le jeu lui-meme vit dans un depot separe :

- **Repo cible** : `knightandwizard-game`
- **Type de produit** : vrai CRPG tactique coop, pas VTT ni simple assistant de table
- **Document fondateur** : [knightandwizard-game-foundation.md](knightandwizard-game-foundation.md)

Les decisions K&W-game ne remplacent pas les decisions K&W documentees dans [../product/](../product/).
